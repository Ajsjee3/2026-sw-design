# EasyTrade 실행 방법

## 프로젝트 구성

```text
android/  - Android 앱 소스 코드
backend/  - Spring Boot API 서버 소스 코드
```

## 서버 주소

백엔드 서버는 아래 주소로 배포되어 있습니다.

```text
https://easytrade-backend-zrxm.onrender.com/
```

Android 앱은 위 서버 주소로 API 요청을 보냅니다.

## 앱 실행

제출한 APK 파일을 Android 스마트폰 또는 Android 에뮬레이터에 설치한 뒤 실행합니다.

```text
EasyTrade_app-debug_final.apk
```

## 확인 순서

```text
회원가입 -> 로그인 -> 005930 검색 -> 종목 상세 화면 -> 선 차트 확인 -> 매수 -> 포트폴리오 확인
```

## 참고

실제 주식 현재가는 한국투자증권 KIS API를 통해 조회합니다.
KIS API 키는 서버 환경변수로 관리하며, GitHub 소스 코드에는 포함하지 않았습니다.
